# StickerApp Backend

Backend for a Barcelona-based print shop that sells custom stickers online. Customers upload images, the frontend generates die-cut + relief masks (OpenCV.js in the browser), and the backend handles users, orders, file storage, and Stripe payments.

Built by **YeKo Studio**.

**Stack**: Python 3.11 · Django 4.2 + DRF · PostgreSQL 15 · Stripe · Docker.
